## 🚀 Quick Start

```bash
git clone https://github.com/LixerDev/AnchorScaffold.git
cd AnchorScaffold
pip install -r requirements.txt
cp .env.example .env
# Optional: add OPENAI_API_KEY for AI-powered generation

# Generate from natural language (AI-powered)
python main.py generate "create a staking program where users earn USDC rewards proportional to time staked"

# Use a specific template
python main.py new staking --name my-staking-program

# List all available templates
python main.py templates

# Describe your program and get the best matching template + code
python main.py generate "I need an escrow where a buyer deposits SOL and the seller ships first"

# Generate with custom program name
python main.py generate "token vesting with 6 month cliff" --name my-vesting --output ./projects
```

## ✨ Two Generation Modes

### Mode 1: Template-Based (Instant)
8 production-ready templates, fully customized with your program name:

| Template | Description | Generated Files |
|---|---|---|
| `staking` | Time-based staking with SPL token rewards | lib.rs, tests |
| `escrow` | Conditional payment escrow with dispute | lib.rs, tests |
| `vesting` | Token vesting with cliff + linear unlock | lib.rs, tests |
| `dao` | Governance: proposals, voting, execution | lib.rs, tests |
| `lottery` | Ticket-based lottery with random winner | lib.rs, tests |
| `multisig` | M-of-N multi-signature wallet | lib.rs, tests |
| `marketplace` | NFT fixed-price marketplace | lib.rs, tests |
| `launchpad` | Fair token launch with SOL contribution | lib.rs, tests |

### Mode 2: AI-Powered (GPT-4)
Describe anything in natural language → AnchorScaffold will:
1. Match to the closest template (if applicable)
2. Use GPT-4 to generate custom program logic
3. Produce valid Anchor syntax with accounts, errors, and events

## 🔍 How Template Matching Works

```
Input: "create a program where users stake JUP and earn DRIFT rewards over time"

Keyword Analysis:
  stake/staking ✅  →  +3 points for 'staking' template
  rewards       ✅  →  +2 points for 'staking' template
  time          ✅  →  +1 point for 'staking' template
  JUP, DRIFT    ✅  →  token names noted (no template impact)

Best match: staking (6 points) → use staking template + AI customization
```

## 🏗️ Architecture

```
main.py (CLI)
    ├── generate <description>  → Parser → Generator → Scaffolder
    ├── new <template>          → Scaffolder directly
    ├── templates               → List all templates
    └── explain <template>      → Detailed template docs

src/
    ├── parser.py          Keyword extraction + intent classification
    ├── generator.py       Template selection + GPT-4 customization
    ├── scaffolder.py      Creates full project directory structure
    ├── test_generator.py  Generates TS tests per template
    └── templates/         8 Anchor/Rust program templates
        ├── staking.py
        ├── escrow.py
        ├── vesting.py
        ├── dao.py
        ├── lottery.py
        ├── multisig.py
        ├── marketplace.py
        └── launchpad.py
```
